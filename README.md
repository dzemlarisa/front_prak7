Фронтенд и бэкенд разработка. 
В процессе выполнения практических работ 7-8 было создано серверное приложение, которое включает в себя следующие маршруты:
POST /api/auth/register - Регистрация пользователя 
POST /api/auth/login - Вход в систему
POST /api/products - Создать товар
GET /api/products - Получить список товаров
GET /api/products/:id - Получить товар по id
PUT /api/products/:id - Обновить параметры товара
DELETE /api/products/:id - Удалить товар  

Также была реализована выдача токена при входе в систему, добавлен защищённый маршрут GET /api/auth/me и защищены маршруты:
GET /api/products/:id
PUT /api/products/:id
DELETE /api/products/:id

В процессе выполнения практических работ 9-10 была реализована генерация refresh-токенов и добавлен маршрут POST /api/auth/refresh, а также реализована фронтенд-часть приложения с использованием React.js и связана с серверной частью.

В практической работе 11 была добавлена система ролей, а также добавлены следующие эндпоинты:
GET /api/users - Получить список пользователей
GET /api/users/:id - Получить пользователя по id
PUT/api/users/:id - Обновить информацию пользователя
DELETE /api/users/:id - Заблокировать пользователя
